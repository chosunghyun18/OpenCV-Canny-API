# OpenCV-Canny-API

Cloud9 - OpenCV Canny API

1. requirements.txt 파일을 만들고 필요한 라이브러리들을 입력

2. 저장 후 pip install -r requirements.txt

3. canny.py 작성

4. 실행 python canny.py
